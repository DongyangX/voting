
# Voting

A voting dapp example using Solana.

## Environment

Node, Rust, Solana-Cli, Anchor

## Install

### Deploy program

Deploy program to Solana `devnet`

```sh
$ anchor build
$ anchor deploy
```

### Start frontend

```sh
$ cd frontend
$ yarn install
$ yarn dev
```
